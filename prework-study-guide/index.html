HTML

  • The head element contains information about the webpage.
  • Metadata falls under the head element, and gives information to the browser about the website.
  • There are parent elements and children elements. Parent elements are elements that define a section of code, while children elements are subsections of those blocks of code.
  • The body element represents the visible content shown to the user.
  • The body can contain children elements such as header, main, and footer.

CSS

  • Three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
    • Inline CSS: add style directly to HTML file by modifying individual elements
    • Internal Style Sheet: contains rules for style in the head section of the HTML file
    • External Style Sheet: separate CSS file that is linked to the HTML file using a link element in the head section
  • CSS Rule = selector {property:value}; {property:value}
    • Each declaration together makes up the declaration block

Git

  • First, clone the remote repository onto your local machine
  • Git flow starts with the main branch. The main branch is the master file of code for our website.
  • To created a new branch:
    • git checkout main
    • git pull origin main
    • git checkout -b feature/branch-name
  • To commit your changes to the remote working branch:
    • git add -A
    • git commit -m "write detailed message"
    • git pull origin main
    • git push origin feature/branch-name)
  • Lastly, once you are confident in your new feature, submit a pull request for team members to review your work before merging the new feature into the main branch.
  • It is best practice to delete feature branches once you are finished working in them.
  • Always switch to and pull the latest version of the main branch to make sure you are up to date with git checkout main and then git pull origin main

JavaScript

  • JavaScript makes the website interactive.
  • Scripts are linked in the body element of the HTML file.
  • Variables are 'objects' in code that contain a value. variable = value
    • Variables are denoted by the keyword var and then the name of the variable.
  • Strings are always surrounded by quotation marks.
  • JavaScript is executed from top to bottom.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
    • Arrays can contain any combination of data types.
    • Arrays are zero-indexed and sequential, meaning the first position in the array is assigned the number 0.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array. Another statement must be used to break out of the code block
    • For loops are constructed as:
      for (expression 1; expression 2; expression 3) {//code block}
      • Expression 1 is executed (one time) before the execution of the code block.
      • Expression 2 defines the condition for executing the code block.
      • Expression 3 is executed (every time) after the code block has been executed.
  • A function is a set of instructions that tells the computer how to perform a certain task.
  • 2 requirements for a function:
    • Define the function
      • Funciton declaration:
        function makefunction () {//code block}
    • Call the function
      • simply write the name of the function:
        makefunction ();
